### 命令格式

cp [选项] <参数1> <参数2>

### 命令功能

和`mv`命令基本相似，但是`cp`用来复制文件或者文件夹的

<!--more-->

### 命令选项

| 选项           | 含义                                       |
| ------------ | ---------------------------------------- |
| `-b`         | 覆盖时，会为其创建一个备份                            |
| `-i`（**推荐**） | 覆盖时，会有交互式提醒                              |
| `-f`         | 覆盖时，不会有任何提醒                              |
| `-r`         | 递归处理，将指定目录及其目录下的所有内容复制到目标位置（**复制目录必须加上该参数**） |
| `-v`         | 详细显示命令的执行过程                              |
| `-u`         | 只会在源文件比目标文件新或者目标文件不存在是，才执行复制操作           |

### 命令参数

| 参数1          | 参数2        | 含义s                    |
| ------------ | ---------- | ---------------------- |
| 源文件          | 目标文件       | 复制+重命名                 |
| 源文件（可以有多个参数） | 目标目录（必须存在） | 复制源文件到目标目录             |
| 源目录          | 目标目录（必须存在） | 复制源目录到目标目录（必须加上`-r`参数） |

以上复制的过程中，若有文件名称重复，则会自动覆盖文件。
